# Liste des sources du corpus

Le corpus repose sur des informations publiques issues des sites officiels des festivals. Les dates, les descriptions éditoriales, les lieux et, lorsque cela était possible, les informations sur les organisateurs et les visuels ont été consolidés à partir de ces sources.

| Festival | Source principale |
|---|---|
| Festival de Cannes | https://www.festival-cannes.com/ |
| Festival d'Avignon | https://festival-avignon.com/ |
| Hellfest | https://hellfest.fr/ |
| Jazz in Marciac | https://www.jazzinmarciac.com/ |
| Festival Interceltique de Lorient | https://www.festival-interceltique.bzh/ |
| Francofolies de La Rochelle | https://www.francofolies.fr/ |
| Montreux Jazz Festival | https://www.montreuxjazzfestival.com/ |

| Glastonbury Festival | https://www.glastonburyfestivals.co.uk/ |
| Edinburgh Festival Fringe | https://www.edfringe.com/ |

## Principe documentaire

- Les sites officiels ont servi de source principale pour les noms, les dates, les lieux et les descriptions courtes.
- Les visuels ont été récupérés depuis les pages ou médias officiels des festivals, puis stockés localement pour éviter le hotlinking.
- Lorsqu'une information n'était pas explicitement publiée, la valeur a été laissée à `NULL` plutôt qu'inventée.
