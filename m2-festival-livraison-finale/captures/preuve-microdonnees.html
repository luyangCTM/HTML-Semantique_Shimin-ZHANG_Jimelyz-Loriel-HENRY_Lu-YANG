Validation du balisage microdonnées

Vue de contrôle produite à partir du HTML final du projet. Elle montre que la page publie bien un type schema.org/Festival et un ensemble cohérent de propriétés itemprop lisibles par un outil d'inspection de microdonnées.

Résumé de validation

Contrôle interne du balisage publié dans la page finale.

Type principal détecté OK - schema.org/Festival
Objets imbriqués OK - Place et Organization
Propriétés principales publiées
name description startDate endDate duration location organizer funder keywords inLanguage image url
Fichier source contrôlé festival-standalone.html

Extrait du HTML publié

Exemple réel d'une carte de festival, nettoyé pour rester lisible dans le rapport.

<article
  id="festival-festival-de-cannes"
  class="festival-card"
  itemscope
  itemtype="https://schema.org/Festival"
  itemid="https://www.festival-cannes.com/"
>
  <div class="card-media">
    <img src="[image intégrée ou locale]"
         alt="Visuel de Festival de Cannes"
         itemprop="image">
  </div>

  <h2 itemprop="name">Festival de Cannes</h2>
  <p itemprop="description">
    Rendez-vous majeur du cinéma mondial...
  </p>

  <time itemprop="startDate" datetime="2025-05-13">13/05/2025</time>
  <time itemprop="endDate" datetime="2025-05-24">24/05/2025</time>
  <meta itemprop="duration" content="P12D">

  <dd itemprop="location" itemscope itemtype="https://schema.org/Place">
    <span itemprop="name">Palais des Festivals et des Congrès</span>
    <span itemprop="addressLocality">Cannes</span>
    <span itemprop="addressCountry">France</span>
  </dd>

  <span itemprop="organizer" itemscope
        itemtype="https://schema.org/Organization">
    <span itemprop="name">Festival de Cannes</span>
  </span>

  <span itemprop="funder" itemscope
        itemtype="https://schema.org/Organization">
    <span itemprop="name">Ville de Cannes</span>
  </span>

  <dd itemprop="keywords">cinéma, compétition, auteur, tapis rouge</dd>
  <dd itemprop="inLanguage">fr,en</dd>
  <a href="https://www.festival-cannes.com/" itemprop="url">
    Site officiel
  </a>
</article>

Cette vue sert de preuve de balisage. Elle montre le HTML réellement publié et les propriétés sémantiques principales, sans prétendre reproduire l'interface d'une extension navigateur spécifique.
